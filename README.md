##  Features

- **Customer Management**
  - Track customer orders
  - Validate customer names 
- **Coffee Products**
  - Maintain coffee menu
  - Validate product names (3+ characters)
- **Order System**
  - Price validation 
  - Relationship management
- ## Analytics
  - Most frequent customers
  - Average coffee prices
  - Order statistics
